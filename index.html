Mācību materiāli

Šie materiāli ir veidoti kā vadlīnijas darbam stundās pedagoga vadībā.
Datorikas pamatu un programmatūras izstrādes apguvē galvenais uzsvars tiek likts uz atvērtā pirmkoda (open source) un bezmaksas rīku izmantošanu, veicinot pieejamību un patstāvīgu prasmju attīstību.

Datorika 9. klasei

Ievads tīmekļa tehnoloģijās un spēļu izstrāde JavaScript valodā.

  • 1. tēma — Tīmekļa struktūra un dizaina domāšana
  • 2. tēma — Stils, multimediji un datņu pārvaldība
  • 3. tēma — Algoritmi un JavaScript pamati
  • 4. tēma — Spēles dzinēja izstrāde un funkcijas
  • 5. tēma — Noslēguma projekta izstrāde un publicēšana

  • 2. tēma — Mikrokontrolētāji un sensori
