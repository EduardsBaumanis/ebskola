6.2 Animācijas un Tween

Tavs šīs stundas izaicinājums: Pievienot animācijas UI un spēles objektiem ar AnimationPlayer un Tween.

SR 2.4.7. UI SR 2.4.15. Modulāras programmas

Teorija: Animācijas Godot

Godot piedāvā divas pieejas animācijām:

  • AnimationPlayer — vizuāli (timeline editor) izveidotas animācijas.
  • Tween — programmatiski izveidotas vienkāršas animācijas.
#include <godot_cpp/classes/tween.hpp>

// Tween — fade in
void show_button() {
    Tween* tween = create_tween();
    button->set_modulate(Color(1, 1, 1, 0));  // sākotnēji caurspīdīgs
    tween->tween_property(button, "modulate", Color(1, 1, 1, 1), 0.5);
}

// Tween — kustība ar bounce
void shake_button() {
    Tween* tween = create_tween();
    Vector2 orig = button->get_position();
    tween->tween_property(button, "position", orig + Vector2(10, 0), 0.05);
    tween->tween_property(button, "position", orig - Vector2(10, 0), 0.05);
    tween->tween_property(button, "position", orig, 0.05);
}

1. uzdevums: Tween animācijas

Programmatiskas animācijas.

Izpildes soļi:

  1. Menu pogas: hover effect ar Tween — palielinās par 1.1x un atgriežas pie release.
  2. Score popup: kad spēlētājs gūst punktus, popup tekst peld augšup un izgaist.
  3. Damage numbers: red text peld no enemy ar Tween.
  4. Visi izmantojot create_tween() un chain() metodes.

2. uzdevums: AnimationPlayer

Timeline editor animācijas.

Izpildes soļi:

  1. Pievieno AnimationPlayer node Sprite2D vecākam.
  2. Izveido animāciju 'walk' ar 4 frame timeline (0.0, 0.25, 0.5, 0.75 sec).
  3. Pievieno frame property keyframes Sprite2D Frame indeksam.
  4. C++ kodā: animation_player->play("walk") when moving.

3. uzdevums: Particle effects

Vizuāli efekti.

Izpildes soļi:

  1. Pievieno GPUParticles2D — explode efekts pie enemy death.
  2. Trail efekts player kustībai.
  3. Coin pickup sparkles.
  4. Konfigurē Inspector: emit count, lifetime, colors.

Papildus uzdevums: Cutscenes

Sarežģītākas animācijas sekvences.

Izpildes soļi:

  1. AnimationPlayer ar multiple tracks: pozīcija, dialogi, kameras zoom.
  2. Spēles starts: 5 sec intro animation.
  3. Boss intro: kameras pan, music swell, name banner.

Biežākās kļūdas

  • Tween nedarbojas: create_tween() kāzas Tween objektu, kas tiek dzēsts; turi referenci.
  • AnimationPlayer skip frames: Animation length pārāk īss vai play() izsaukts vairākkārt.
  • Particles invisible: CollisionShape2D vai Sprite2D parent var ietekmēt visibility.

Godot ekrānuzņēmumi

Koda piemērs (paplašināts)

#include <godot_cpp/classes/tween.hpp>

void Player::on_take_damage(int amount) {
    // Damage number popup
    Label* dmg = memnew(Label);
    dmg->set_text("-" + String::num_int64(amount));
    dmg->set_position(get_position() + Vector2(0, -30));
    dmg->add_theme_color_override("font_color", Color(1, 0, 0));
    get_parent()->add_child(dmg);

    // Tween: peld augšup un izgaist
    Tween* tween = dmg->create_tween();
    tween->set_parallel();  // abas animācijas paralēli
    tween->tween_property(dmg, "position",
                          dmg->get_position() + Vector2(0, -50), 0.8);
    tween->tween_property(dmg, "modulate:a", 0.0, 0.8);
    tween->chain()->tween_callback(Callable(dmg, "queue_free"));

    // Player flash red
    Tween* flash = create_tween();
    set_modulate(Color(1, 0, 0));
    flash->tween_property(this, "modulate", Color(1, 1, 1), 0.3);
}
Bojājot ienaidnieku, parādās '-25' sarkanā tekstā, peld augšup un izgaist; player sarkans flash 0.3 sec.
