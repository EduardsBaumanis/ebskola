# Programmēšana II — Godot ekrānuzņēmumi

Šajā mapē jāievieto Godot editor un spēles ekrānuzņēmumi, kas tiek atsauktas
lessoni HTML failos. Katrs attēls ir aprakstīts ar `alt` atribūtu attiecīgajā
`<img>` tagā.

## Standarta nosaukumu konvencija
- `topic{N}_{nosaukums}.png` — kur N ir tēmas numurs (1–6).

Piemērs: `topic1_godot_pm.png`, `topic2_player_moving.png`, ...

Attēlus jāveido 1280×720 vai augstākā izšķirtspējā, PNG vai WebP formātā.
