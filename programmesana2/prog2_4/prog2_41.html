4.1 Vector un Array

Tavs šīs stundas izaicinājums: Saprast atšķirību starp std::vector un Godot Array, izvēlēties piemēroto datu struktūru.

SR 2.4.16. Saliktas datu struktūras SR 2.4.13. Datu tipi

1. uzdevums: Enemy spawning ar vector

Pārvaldi vairākus ienaidniekus.

Izpildes soļi:

  1. Game klasē: std::vector<Enemy*> enemies.
  2. Funkcija spawn_enemy(): izveido Enemy, pievieno scēnai (add_child), pievieno vektoram.
  3. _process: ar timeru ik pa 2 sec spawno jaunu ienaidnieku.
  4. Pārbaudi, vai visi tiek pareizi pievienoti.

2. uzdevums: Iteration un dzēšana

Apstaigā un noņem mirušos.

Izpildes soļi:

  1. For cikls iziet caur enemies, izsauc e->update().
  2. Atsevišķā ciklā vai erase-remove idiom: noņem mirušos.
  3. Ievēro: nedrīkst modificēt vektoru cikla laikā ar parastu for.
  4. Lieto: enemies.erase(std::remove_if(...), enemies.end());

3. uzdevums: Array vs vector eksperiments

Salīdzini abas pieejas.

Izpildes soļi:

  1. Vienu no projektiem realizē ar std::vector, otru ar Godot Array.
  2. Mēri laiku 1000 elementu pievienošanai un nolasīšanai.
  3. Pieraksti rezultātus README.
  4. Secinājums: vector ātrāks, Array elastīgāks ar Variant.

Papildus uzdevums: Reserve un capacity

Vector veiktspējas optimizācija.

Izpildes soļi:

  1. enemies.reserve(100); pirms cikla — alocē atmiņu, izvairās no relok.
  2. Pārbaudi: enemies.capacity() vs enemies.size().
  3. Ar daudziem objektiem reserve ir kritisks veiktspējai.

Biežākās kļūdas

  • Crash with begin(): Vector tukšs, bet mēģini iegūt elements.
  • Memory leak: 'new Enemy' bez delete. Lieto unique_ptr vai add_child.
  • Iterator invalidation: Pievienojot/dzēšot vektoru cikla laikā, iterator var kļūt nederīgs.

Godot ekrānuzņēmumi

Koda piemērs (paplašināts)

std::vector<Enemy*> enemies;

void spawn_enemy() {
    Enemy* e = memnew(Enemy);
    add_child(e);
    e->set_position(Vector2(randf_range(0, 1280), 0));
    enemies.push_back(e);
}

void Game::_physics_process(double delta) {
    spawn_timer += delta;
    if (spawn_timer >= 2.0) {
        spawn_enemy();
        spawn_timer = 0;
    }
    // Noņem mirušos
    enemies.erase(
        std::remove_if(enemies.begin(), enemies.end(),
            [](Enemy* e) { return !e->is_alive(); }),
        enemies.end());
}
Spēle spawno 1 ienaidnieku ik pa 2 sec, mirstīgi tiek noņemti no vektora un scēnas.
