4.6 Noslēguma projekts: Top-down šautene

Tavs šīs stundas izaicinājums: Apvienot datu struktūras, AI un object pooling pilnvērtīgā top-down šautenē.

SR 2.4.1. Problēmu dekompozīcija SR 2.4.16. Datu struktūras SR 2.4.18. OOP SR 2.4.19. Algoritmu efektivitāte

Teorija: Spēles apraksts un arhitektūra

Top-down šautene ar šādām funkcijām:

  • Spēlētājs ar 8 virzienu kustību un mouse-aimed šaušanu.
  • 3 ienaidnieku tipi ar dažādu AI uzvedību.
  • Pathfinding ap šķēršļiem.
  • Wave system: ienaidnieku viļņi ar augošu sarežģītību.
  • Pickup sistēma: HP, ammo, power-ups.
  • Score un wave skaitītājs.
// Klašu hierarhija
class Player : public CharacterBody2D { /* WASD + shoot */ };
class Enemy : public CharacterBody2D { /* state machine + pathfinding */ };
class Goblins : public Enemy { /* fast, weak */ };
class Brute : public Enemy { /* slow, strong */ };
class Shooter : public Enemy { /* keeps distance, shoots */ };
class Bullet : public Area2D { /* pooled */ };
class WaveManager : public Node { /* spawns waves */ };

1. uzdevums: Pamata struktūra (1.)

Player + Enemy + Bullet pool.

Izpildes soļi:

  1. Pārkopē Player no 2.2 stundas, pievieno mouse-aim.
  2. Enemy no 4.3 ar state machine.
  3. Bullet pool no 4.5.
  4. 1 līmeņa scēna ar 5 ienaidniekiem un sienām.

2. uzdevums: Vairāki ienaidnieku tipi (2.)

Polimorfisms ar Enemy hierarhiju.

Izpildes soļi:

  1. Goblins: speed 200, hp 20, melee.
  2. Brute: speed 80, hp 100, melee, 2x bojājums.
  3. Shooter: speed 100, hp 40, ranged — saglabā distanci 200, šauj.
  4. Pievieno katra tipa Sprite ar dažādām krāsām.

3. uzdevums: Wave sistēma un balanss (3.)

Pielāgojama sarežģītība.

Izpildes soļi:

  1. WaveManager klase ar std::vector<Wave> waves.
  2. Wave struct: {int goblins; int brutes; int shooters;}
  3. Ik pa 30 sec uzsāk nākamo wave.
  4. UI: "Wave 3 — 12 enemies remaining".
  5. Boss after wave 5: 500 HP, multiple attack patterns.
  6. GitHub: 8+ commit, README ar gameplay GIF.

Papildus uzdevums: Bonus: Power-ups un upgrades

Spēles dziļums.

Izpildes soļi:

  1. Pickups: HP+50, Ammo+20, ShieldStrong (15 sec invulnerable).
  2. Upgrade tree: izvēlies starp +damage, +speed, +max HP starp wave.
  3. Saglabā augstāko score JSON failā.

Biežākās kļūdas

  • Lag ar daudziem enemies: Pārliecinies, ka NavigationAgent2D update_freq ir saprātīgs.
  • Bullets sit savus: Pārbaudi collision layers — Player bullets jāignore Player.
  • Wave neuzsākas: Timer vai counter sajaucies — debug ar print statements.

Godot ekrānuzņēmumi

Koda piemērs (paplašināts)

class WaveManager : public Node {
    GDCLASS(WaveManager, Node)

private:
    int current_wave = 0;
    float wave_timer = 0;
    std::vector<Enemy*> active_enemies;

    struct Wave { int goblins, brutes, shooters; };
    std::vector<Wave> waves = {
        {3, 0, 0},      // wave 1
        {5, 1, 0},      // wave 2
        {6, 2, 1},      // wave 3
        {8, 3, 2},      // wave 4
        {10, 5, 3}      // wave 5 + boss
    };

public:
    void _process(double delta) override {
        wave_timer -= delta;
        if (active_enemies.empty() && wave_timer <= 0) {
            start_next_wave();
        }
    }

    void start_next_wave() {
        if (current_wave >= waves.size()) {
            UtilityFunctions::print("All waves complete!");
            return;
        }
        Wave w = waves[current_wave];
        for (int i = 0; i < w.goblins; i++) spawn(EnemyType::GOBLIN);
        for (int i = 0; i < w.brutes; i++) spawn(EnemyType::BRUTE);
        for (int i = 0; i < w.shooters; i++) spawn(EnemyType::SHOOTER);
        current_wave++;
        wave_timer = 30.0f;
    }
};
Pilnvērtīga top-down šautene ar 5 viļņiem, 3 ienaidnieku tipiem, pathfinding un boss.
