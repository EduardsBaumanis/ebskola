1.5 Spēles laukuma izveide

Tavs šīs stundas izaicinājums: Izveidot Pong spēles laukuma vizuālo izkārtojumu un sagatavot to spēles loģikai.

SR 2.4.7. Lietotāja saskarne (UI) SR 2.4.13. Datu tipi

Teorija: Coordinates (Koordinātes) un transformācijas

Godot 2D koordinātes sākas kreisajā augšējā stūrī: (0, 0). X pieaug pa labi, Y uz leju.

// Pozīciju manipulācija C++
Vector2 pos = get_position();
pos.x += 10;             // pa labi par 10 pikseļiem
pos.y -= 5;              // uz augšu par 5 pikseļiem
set_position(pos);

set_position(Vector2(640, 360));  // ekrāna centrs 1280x720

Transform = Position + Rotation + Scale. Bērnu nodes pārmanto vecāku transform.

Viewport (Skatlogs) = loga apgabals, kurā renderē scēnu. Standarta Pong: 1280×720.

1. uzdevums: Project Settings

Iestati spēles izmēru.

Izpildes soļi:

  1. Project → Project Settings.
  2. Display → Window: Viewport Width 1280, Height 720.
  3. Stretch Mode: viewport, Aspect: keep.
  4. Application → Config → Name: Pong.
  5. Run → Main Scene: main.tscn.

2. uzdevums: Sienas un dalītājs

Pievieno laukuma elementus.

Izpildes soļi:

  1. Pievieno Line2D kā Main bērnu, pārsauc CenterLine.
  2. Inspector → Points: divi punkti (640, 0), (640, 720).
  3. Width: 4, Default Color: balta.
  4. Pievieno divus StaticBody2D ar CollisionShape2D — TopWall un BottomWall.
  5. Pozīcijas (640, 10) un (640, 710), formas izmērs 1280×20.

3. uzdevums: Punktu skaitītāji

Label nodes punktiem.

Izpildes soļi:

  1. Pievieno Label kā ScoreLeft. Text: 0, Font Size: 64, Position: (300, 50).
  2. Klonē uz ScoreRight, Position: (920, 50).
  3. Pievieno Title Label: Text: PONG, Position: (560, 30), Font Size: 32.
  4. Pārbaudi izvietojumu, palaižot.

Papildus uzdevums: Stila uzlabojumi

Pievieno krāsas un kameru.

Izpildes soļi:

  1. Pievieno Camera2D kā Main bērnu.
  2. Position: (640, 360), Make Current: true.
  3. Visiem Sprite2D Modulate: balts.

Biežākās kļūdas

  • Spēle palaižas baltā logā: Main Scene Project Settings nav iestatīta.
  • Objekti aizsedz Background: Pārbaudi Z-index Inspector.
  • Camera2D nedarbojas: Make Current Inspector vai current = true kodā.

Godot ekrānuzņēmumi

Koda piemērs (paplašināts)

[node name="ScoreLeft" type="Label" parent="."]
position = Vector2(300, 50)
text = "0"
theme_override_font_sizes/font_size = 64

[node name="CenterLine" type="Line2D" parent="."]
points = PackedVector2Array(640, 0, 640, 720)
width = 4.0

[node name="Camera2D" type="Camera2D" parent="."]
position = Vector2(640, 360)
