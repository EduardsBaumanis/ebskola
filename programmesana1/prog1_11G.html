Palīgmateriāli

1. tēma: Ievads un darba vide. 1. stunda: Datora sagatavošana darbam.

Tavs šīs stundas izaicinājums: Tu iemācīsies atpazīt galvenos programmētāja darba rīkus, patstāvīgi sagatavot savu datoru darbam un palaist savu pirmo kodu, rūpējoties par savu veselību un drošību.

SR 2.4.12. (Vide), 2.3.9. (Datora resursi), 2.3.11. (Drošība), 3.1.1. (Ergonomika)

1. uzdevums (Vienkāršs): Programmu uzstādīšana un pirmais tests

Sagatavot datoru programmēšanai un pārliecināties, ka tas saprot Python valodu.

  1. Atver Microsoft Store lietotni savā datorā.
  2. Meklēšanas joslā ieraksti "Python 3.12" un nospied "Iegūt" (Get) vai "Instalēt".
  3. Tajā pašā veikalā meklē "Visual Studio Code" un uzinstalē to.
  4. Atver Visual Studio Code, spied File -> New Text File.
  5. Ieraksti: print("Sveika, pasaule!").
  6. Saglabā failu ar nosaukumu tests.py uz sava darbvirsmas (Desktop).
  7. Nospied mazo trīsstūra pogu (Run) loga augšējā labajā stūrī.

2. uzdevums (Viduvējs): Kārtības ieviešana ar komandrindu

Iemācīties izmantot termināli (teksta lodziņu), lai izveidotu mapes savam projektam.

  1. Programmas VS Code apakšā atrodi un atver sadaļu Terminal.
  2. Ieraksti komandu mkdir mana_spele un nospied Enter (tā izveidos jaunu mapi).
  3. Ieraksti cd mana_spele, lai ieietu šajā mapē.
  4. Izveido vēl vienu mapi kodam, rakstot mkdir kods.
  5. Pārvelc savu iepriekš izveidoto failu tests.py uz mapi kods.

3. uzdevums (Padziļināts): Projekta plānošana

Pievienot palīgrīkus ērtākam darbam un uzrakstīt spēles ideju.

  1. VS Code kreisajā sānā uzmeklē pogu Extensions (izskatās pēc četriem kvadrātiņiem).
  2. Meklētājā ieraksti "Python" un uzinstalē pirmo rezultātu no Microsoft. Tas palīdzēs datoram labāk Tevi saprast.
  3. Izveido jaunu failu ar nosaukumu apraksts.txt.
  4. Šajā failā uzraksti divus teikumus: kādu spēli Tu gribētu izveidot un kāds būs tās galvenais varonis.
  5. Saglabā šo failu mapē mana_spele.

Jaunās vielas kopsavilkums

  • Programmu iegūšana: Skolā izmantojam tikai Microsoft Store, jo tas ir vienīgais drošais veids, kā ielādēt pārbaudītas programmas.
  • Darba vide (IDE): Visual Studio Code ir kā tavs digitālais darbagalds – šeit mēs rakstīsim un labosim kodu.
  • Valodas dzinējs: Python ir programma, kas pārvērš tavu rakstīto tekstu darbībās, ko saprot dators.
  • Terminālis: Īpašs lodziņš, kurā mēs dodam datoram tiešas komandas ar tekstu, nevis klikšķinot ar peli.
  • Ergonomika: Sēdi taisni, turi ekrānu acu līmenī un ik pēc 20 minūtēm atpūtini acis, skatoties tālumā.

Zini, kāpēc tas ir svarīgi? Ja dators būs nekārtīgs, tajā būs grūti strādāt. Sākot ar kārtību, Tu ietaupi laiku un pasargā savu darbu no pazušanas.
