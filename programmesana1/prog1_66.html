6. tēma: Noslēguma darbs. Projekts: "CYOA: Pasaule no faila"

Apsveicu! Tu esi nonācis līdz brīdim, kad Tavs kods kļūst par universālu dzinēju. Tavs uzdevums ir izveidot spēli, kurā neviens stāsta teikums neatrodas Python failā — viss tiek ielādēts dinamiski no CSV, bet spēlētāju sasniegumi tiek analizēti un glabāti JSON formātā.

Tavs mērķis: Radīt programmu, kas strādātu ar jebkuru stāsta failu, ja vien tam ir pareiza struktūra.

Tehniskās prasības (Checklist)

  • Datu nodalīšana: Viss stāsts (teksti, izvēles, ID) tiek ielādēts no pasaule.csv.
  • Viedā karte: CSV dati tiek transformēti vārdnīcā {ID: dati} ātrai piekļuvei.
  • Navigācijas loģika: Spēlētājs pārvietojas pa stāstu, izmantojot mērķa ID (merkis_a, merkis_b).
  • Pastāvīgā atmiņa: Spēles beigās rezultāti tiek pievienoti leaderboard.json failam.
  • Analītika: Spēles sākumā vai beigās tiek parādīta statistika (piem., vidējais punktu skaits).

Snieguma līmeņa apraksts (SLA)

Kritērijs Sācis apgūt (1-3) Turpina apgūt (4-6) Apguvis (7-8) Padziļināti (9-10)
Datu ielāde Nolasa TXT failu. Nolasa CSV, bet neizmanto ID sistēmu. Dinamiski ielādē CSV un transformē to vārdnīcā. Ieviesta kļūdu apstrāde (try-except) bojātiem failiem.
Datu pastāvība Dati pazūd pēc izslēgšanas. Saglabā JSON, bet nepapildina esošo sarakstu. Lieto Read-Modify-Write ciklu JSON failiem. Sagatavo datus eksportam un vizualizācijai.
Kvalitāte Kodā ir stāsta teksti. Kods ir vienā lielā failā. Kods sadalīts moduļos (main, engine, data). Profesionāls README un tīra Git vēsture.

Darba gaita

  1. Datu plānošana: Eksportē savu Google Sheets tabulu kā pasaule.csv.
  2. Dzinēja izveide: Uzraksti funkciju, kas pārvērš CSV par "kartes" vārdnīcu.
  3. Spēles cikls: Izmanto while ciklu, kas nolasa pašreizējo lokāciju un gaida lietotāja izvēli.
  4. Rezultātu uzskaitne: Pievieno JSON saglabāšanas loģiku un statistikas aprēķinu.
  5. Analīze: Aprēķini vidējo rādītāju: $$vidējais = \frac{\sum punktu\_summa}{n}$$

ebSkola Mentora padomi:

1. Modularitāte: Turiet main.py tīru — tajā jābūt tikai spēles ciklam. Visu ielādi lieciet engine.py.
2. README faili: Labs programmētājs paskaidro, kā lietot viņa dzinēju. Pievieno instrukciju, kādai jābūt CSV struktūrai!
3. Git Commits: Neveic vienu lielu push beigās. Rādi savu progresu soli pa solim. 

import csv

def ieladet_pasauli(faila_vards):
    karte = {}
    with open(faila_vards, mode='r', encoding='utf-8') as f:
        reader = csv.DictReader(f)
        for rinda in reader:
            loc_id = int(rinda['id'])
            karte[loc_id] = rinda
    return karte
