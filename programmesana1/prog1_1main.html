1. Tēma — Ievads programmēšanā, izstrādes vide un versiju pārvaldība

Par šo tēmu

Šī tēma veido pamatu visam turpmākajam kursam. Tās mērķis ir iemācīt ne tikai rakstīt pirmo Python programmu, bet arī strādāt strukturēti un profesionāli. Tu apgūsi izstrādes vides uzstādīšanu, komandrindas lietošanu un projekta mapju struktūras veidošanu.

Tēmas laikā tiek ieviesti versiju pārvaldības pamati ar Git: repozitorija izveide, commit veidošana, zaru (branch) izmantošana un .gitignore nozīme. Tu iemācīsies saprast, kā tiek saglabāta izmaiņu vēsture un kā droši eksperimentēt ar kodu.

Papildus programmēšanas videi mēs apskatīsim operētājsistēmas resursus (CPU, RAM), darba efektivitātes principus un ergonomiku. Tēmas noslēgumā Tu izstrādāsi nelielu CLI (komandrindas) mini-projektu, pielietojot projektu struktūru, Git darba plūsmu, dokumentēšanu un drošības principus.

Tēmas saturs (Stundu plāns)

  • 1. stunda: Izstrādes vide un pirmais projekts
  • 2. stunda: Komandrinda un pamata drošība
  • 3. stunda: Git pamati un versiju pārvaldība
  • 4. stunda: OS resursi un darba efektivitāte
  • 5. stunda: Mini-projekts (CLI aplikācija)
  • 6. stunda: Noslēguma pārbaudes darbs
