Tēmas Sasniedzamie rezultāti Palīgmateriāli

4. tēma: Funkcijas un Modularitāte. 2. stunda: Scope un return.

Tavs šīs stundas izaicinājums: Tu iemācīsi funkcijām "sarunāties" ar pārējo programmu. Tu apgūsi, kā izmantot return, lai saņemtu aprēķinu rezultātus, un uzzināsi, kāpēc daži mainīgie ir "slepeni" un pieejami tikai funkcijas iekšienē.

SR 2.4.15. (Atgriežamās vērtības un lokālie mainīgie)

Teorija: Svarīgākie noteikumi

  • Return pret Print: print() tikai parāda tekstu ekrānā. return "izmet" vērtību no funkcijas ārā, lai to varētu saglabāt mainīgajā un lietot tālāk.
  • Lokālie mainīgie (Scope): Mainīgie, kas definēti funkcijas iekšpusē, ir "dzīvi" tikai tajā. Ārpusē tie neeksistē.
  • Funkcijas beigas: Tiklīdz dators sasniedz komandu return, funkcijas darbs tiek pārtraukts. Jebkurš kods zem tās netiks izpildīts.

Praktiskie uzdevumi

1. uzdevums: Zāļu eliksīrs (return)

  1. VS Code izveido failu eliksirs.py.
  2. Definē funkciju izrekinat_veselibu(pasreizeja, eliksira_speks):.
  3. Funkcijas iekšienē saskaiti abus skaitļus.
  4. Tā vietā, lai izmantotu print(), ieraksti: return summa.
  5. Ārpus funkcijas izsauc to un saglabā rezultātu mainīgajā: jauna_veseliba = izrekinat_veselibu(50, 20).
  6. Izvadi galarezultātu lietotājam: "Pēc eliksīra lietošanas Tev ir 70 HP!".

2. uzdevums: Lokālais "slazds" (Scope)

  1. Izveido failu scope_tests.py.
  2. Definē funkciju pievienot_zeltu():.
  3. Funkcijas iekšienē izveido mainīgo zelts = 100.
  4. Izsauc funkciju un tūlīt pēc tam mēģini izpildīt print(zelts) ārpus funkcijas.
  5. Palaid programmu un novēro NameError kļūdu.
  6. Labojums: Pārveido funkciju, lai tā atgrieztu vērtību (return), un piešķir to mainīgajam ārpusē.

3. uzdevums: CYOA Izvēles dzinējs

  1. Izveido failu izveles_dzinejs.py.
  2. Definē funkciju veikt_izveli(jautajums, opcija_a, opcija_b):.
  3. Funkcijas iekšienē izvadi jautājumu, opcijas un palūdz lietotājam ievadi ("A" vai "B").
  4. Izmanto return, lai "atdotu" ievadīto burtu spēles galvenajai daļai.
  5. Izsauc funkciju mainīgajā cels = veikt_izveli(...) un ar if pārbaudi: ja rezultāts ir "A", izvadi "Tu dodies uz mežu", ja "B" — "Tu dodies uz pili".

Biežākās kļūdas:

1. "Pazudusī" vērtība: Tu izmanto return, bet ārpusē nepiešķir rezultātu mainīgajam. Tad vērtība vienkārši izgaist.
2. Scope kļūda: Mēģinājums piekļūt mainīgajam, kas "pieder" funkcijai, no ārpuses.
3. None tips: Ja funkcijā trūkst return, bet Tu mēģini rezultātu saglabāt, mainīgais saņems tukšu vērtību None.

Koda paraugs

def dabu_punktus():
    bonuss = 50 # Lokālais mainīgais
    return bonuss

# Ārpusē 'bonuss' neeksistē, mums ir jāizmanto funkcijas rezultāts
rezultats = dabu_punktus()
print(f"Tu ieguvi {rezultats} punktus!")

# Kods pēc return nekad netiek izpildīts:
def beigas():
    return "Pirmais ziņojums"
    print("Šo rindu dators ignorēs!")
Tu ieguvi 50 punktus!
