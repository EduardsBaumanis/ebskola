SR 2.4.13. (Sintakse, datu tipi, IO)

1. uzdevums (Vienkāršs): Duelanta anketa

Iemācīties iegūt vairākus teksta datus un tos apvienot loģiskā izvades teikumā.

Izpildes soļi:

  1. VS Code izveido failu anketa.py.
  2. Izveido mainīgo vards un piešķir tam vērtību no input("Kā sauc Tavu varoni? ").
  3. Izveido mainīgo speja un pajautā lietotājam: "Kāda ir viņa superspēja?".
  4. Izvadi terminālī aprakstu: print("Varonis", vards, "gatavojas izmantot spēju:", speja).
  5. Pievieno trešo jautājumu par varoņa vecumu un iekļauj to izvades teikumā.

2. uzdevums (Viduvējs): Veselības punktu (HP) kalkulators

Praktiski pielietot veselu skaitļu (int) konvertāciju un matemātisku saskaitīšanu.

Izpildes soļi:

  1. Izveido failu veseliba.py.
  2. Izveido mainīgo sakuma_hp, kurā lietotājs ievada skaitli (piemēram, 100).
  3. Izveido mainīgo bonuss, kurā lietotājs ievada bruņu punktus (piemēram, 25).
  4. Svarīgi: Abas ievades "iesaiņo" int() funkcijā, lai tās kļūtu par skaitļiem.
  5. Saskaiti abus mainīgos: kopejais = sakuma_hp + bonuss.
  6. Izvadi ziņojumu: print("Tavs duelētājs sāk cīņu ar", kopejais, "HP").

3. uzdevums (Padziļināts): Veiksmes koeficients

Darbs ar daļskaitļiem (float) un programmas "salūšanas" analīze.

Izpildes soļi:

  1. Izveido failu veiksme.py.
  2. Palūdz ievadīt "Uzbrukuma spēku" (konvertē uz int).
  3. Palūdz ievadīt "Veiksmes reizinātāju" (konvertē uz float, piemēram, 1.15).
  4. Sareizini abus mainīgos un izvadi gala rezultātu.
  5. Eksperiments: Palaid programmu un pie reizinātāja ieraksti tekstu (piemēram, "daudz"). Novēro kļūdu terminālī.

Svarīgākie noteikumi

  • Interaktivitāte: Programma vairs nav statiska; tā apstājas un gaida lietotāja ievadi.
  • Zelta likums: input() funkcija visus datus "atdod" kā tekstu (str).
  • Tulkāšana:
    • int() — ja vēlies veselu skaitli;
    • float() — ja vēlies skaitli ar punktu.

Biežākās kļūdas:

1. ValueError: Tu mēģini pārvērst burtus par skaitli. Python to neprot. Pārbaudi, vai ievadi tikai ciparus!
2. TypeError: Tu mēģini saskaitīt tekstu ar skaitli (piemēram, "Punkti: " + 10). Izmanto komatu, lai tos atdalītu: print("Punkti:", 10). 

# Pareiza datu iegūšana duelītim:
vards = input("Ieraksti vārdu: ")
dzivibas = int(input("Ieraksti dzīvības: "))

# Ja lietotājs ievada 50, tad 'dzivibas' ir skaitlis, nevis teksts.
print(vards, "dodas cīņā ar", dzivibas, "HP!")
