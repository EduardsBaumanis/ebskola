Teorija: Streamlit — Python kā web saskarne

Streamlit ir Python ietvars, kas pārvērš parastu Python kodu par interaktīvu web lietotni — bez HTML/CSS/JS rakstīšanas.

import streamlit as st

# Virsraksti
st.title("Mans projekts")
st.header("Apakšsadaļa")
st.write("Vienkāršs teksts")
st.markdown("**Treknrakstā** un *kursīvā*")

# Ievades elementi
vards = st.text_input("Tavs vārds:")
vecums = st.number_input("Vecums", min_value=1, max_value=120, value=18)
ir_skoleens = st.checkbox("Es esmu skolēns")
limenis = st.slider("Grūtība", 1, 10, 5)

if st.button("Sākt"):
    st.success(f"Sveiks, {vards}! ({vecums} g.)")

# Datu attēlošana
import pandas as pd
df = pd.DataFrame({"Vārds": ["Anna", "Jānis"], "Punkti": [120, 95]})
st.dataframe(df)
st.bar_chart(df.set_index("Vārds"))

Palaiž ar streamlit run app.py. Lapa pati pārlādējas, kad maini kodu!

1. uzdevums: Pirmā Streamlit lapa

Izveido vienkāršu sveiciena lietotni.

Izpildes soļi:

  1. pip install streamlit.
  2. Izveido app.py ar st.title(), st.write(), st.text_input().
  3. Palaiž: streamlit run app.py.
  4. Atvēr pārlūkā http://localhost:8501.
  5. Pievieno st.button() un parādi paziņojumu pēc klikšķa.

2. uzdevums: Skaitļu minētājs Streamlit

Pārvērt iepriekšējo terminālī skriptu par web lietotni.

Izpildes soļi:

  1. Pārvieto Skaitļu minētāja loģiku no terminala uz Streamlit.
  2. Lieto st.session_state spēles stāvokļa glabāšanai (session_state.skaitlis).
  3. st.number_input() minējumam, st.button("Min") apstiprināšanai.
  4. Atgriezi st.success() vai st.warning() atkarībā no rezultāta.
  5. Pievieno "Jauna spēle" pogu, kas resetē session_state.

3. uzdevums: Diagrammas un dati

Pievieno datu vizualizāciju.

Izpildes soļi:

  1. Izveido highscore.csv ar 10 spēlētāju rezultātiem.
  2. import pandas as pd; df = pd.read_csv("highscore.csv").
  3. st.dataframe(df) — interaktīva tabula.
  4. st.bar_chart(df.set_index("vards")).
  5. Pievieno filtru: limit = st.slider("Cik rādīt", 1, 10, 5); df.head(limit).

Biežākās kļūdas

  • Lietotne pārlādējas no nulles: Streamlit pārpalaiž visu skriptu pie katras izmaiņas. Lieto session_state.
  • Streamlit komandas ārpus skripta: st.write u.c. strādā tikai streamlit run kontekstā.
  • Aizmirsts if st.button(): bloks: Bez tā kods izpildās uzreiz.

Koda piemērs

import streamlit as st
import random

st.title("▸ Skaitļu minētājs")

if "skaitlis" not in st.session_state:
    st.session_state.skaitlis = random.randint(1, 100)
    st.session_state.megi = 0

minejums = st.number_input("Min skaitli (1-100):", 1, 100, 50)

if st.button("Min!"):
    st.session_state.megi += 1
    if minejums == st.session_state.skaitlis:
        st.success(f"★ Trāpīts {st.session_state.megi}. mēģinājumā!")
    elif minejums < st.session_state.skaitlis:
        st.info("↑ Patiesais ir lielāks")
    else:
        st.info("↓ Patiesais ir mazāks")

if st.button("Jauna spēle"):
    del st.session_state.skaitlis
    st.rerun()
