9.1 Ārējās bibliotēkas un instalēšana

Tavs šīs stundas izaicinājums: Iepazīt Python bibliotēku ekosistēmu un iemācīties instalēt un izmantot ārējās bibliotēkas (pip).

SR 2.4.11. Lieto standartizētas bibliotēkas un API SR 2.4.12. Sagatavo programmēšanas vidi

Teorija: pip un virtual environment

Bibliotēka ir cita programmētāja jau uzrakstīts kods, ko vari iekļaut savā projektā ar vienu komandu. Python ir aptuveni 500,000 publisku bibliotēku repozitorijā PyPI.

# Instalācija
pip install streamlit
pip install requests
pip install --upgrade pillow

# Saraksts ar instalētajām
pip list

# Saglabā atkarības
pip freeze > requirements.txt
pip install -r requirements.txt

Virtual environment ir izolēta vide projektam, lai dažādi projekti varētu izmantot dažādas versijas:

python -m venv venv          # izveido
venv\Scripts\activate         # Windows aktivizācija
source venv/bin/activate     # Linux/Mac aktivizācija
deactivate                   # iziet

1. uzdevums: Pirmā instalācija

Sagatavo darba vidi ar virtual environment.

Izpildes soļi:

  1. Termināli: python -m venv venv.
  2. Aktivizē: venv\Scripts\activate (Windows) vai source venv/bin/activate (Linux).
  3. Pārbaudi: which python — jārāda venv mapē.
  4. Instalē: pip install requests.
  5. Pārbaudi: pip list.

2. uzdevums: Requests bibliotēka

Izmanto pirmo bibliotēku praktiski.

Izpildes soļi:

  1. Izveido laiks.py.
  2. import requests; r = requests.get("https://api.open-meteo.com/v1/forecast?latitude=56.95&longitude=24.10¤t_weather=true").
  3. Izvada r.json() — laika dati par Rīgu.
  4. Pievieno kļūdu apstrādi: try/except requests.RequestException.

3. uzdevums: requirements.txt

Saglabā un atjauno atkarības.

Izpildes soļi:

  1. pip freeze > requirements.txt.
  2. Apskati saturu — kuras bibliotēkas un kādas versijas.
  3. Eksperiments: dzēš venv mapi.
  4. Atjauno: python -m venv venv; aktivizē; pip install -r requirements.txt.
  5. Pārliecinies, ka kods strādā.

Papildus uzdevums: Streamlit "Hello World"

Pirmā tīmekļa lietotne 5 rindās.

Izpildes soļi:

  1. pip install streamlit.
  2. app.py: import streamlit as st; st.title("Mans pirmais web"); st.write("Sveiks!").
  3. Palaiž: streamlit run app.py.
  4. Pievieno st.button("Klikšķini") un st.text_input().

Biežākās kļūdas

  • Aizmirsts aktivizēt venv: Bibliotēkas instalējas globāli, sajaucot dažādu projektu atkarības.
  • pip vs pip3: Daudzās sistēmās jālieto pip3 Python 3 versijai.
  • requirements.txt nav GitHub: Bez tā citi nevar palaist tavu projektu.

Koda piemērs

import requests

def laiks_riga():
    r = requests.get(
        "https://api.open-meteo.com/v1/forecast",
        params={"latitude": 56.95, "longitude": 24.10, "current_weather": True}
    )
    if r.status_code != 200: return None
    return r.json()["current_weather"]

dati = laiks_riga()
if dati:
    print(f"Rīgā: {dati['temperature']}°C, vējš {dati['windspeed']} km/h")
Rīgā: 5.4°C, vējš 12 km/h
