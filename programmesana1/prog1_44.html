Tavs šīs stundas izaicinājums: Iemācīties rakstīt kodu, kas ir ne tikai funkcionāls, bet arī skaists un lasāms. Tu apgūsi PEP 8 — pasaulē atzītus Python "etiķetes noteikumus", lai Tavs kods būtu saprotams ikvienam programmētājam pasaulē.

SR 2.4.8. (Koda noformēšanas kultūra)

1. uzdevums (Vienkāršs): "Briesmīgā koda" glābšana

Iemācīties saskatīt stila kļūdas un tās manuāli izlabot, neizmainot programmas loģiku.

Izpildes soļi:

  1. VS Code izveido failu haoss.py.
  2. Iekopē tajā kodu, kurā nav atstarpju, ir slikti mainīgo nosaukumi (piem., x1, fnc) un funkcijas "salipušas" kopā.
  3. Pārveido nosaukumus uz snake_case (mazie burti ar pasvītrojumu).
  4. Saliec atstarpes ap operatoriem un divas tukšas rindas starp funkcijām.
  5. Palaid programmu, lai pārliecinātos, ka pēc "tīrīšanas" tā joprojām strādā.

2. uzdevums (Viduvējs): Automātiskais stilists

Iepazīties ar VS Code rīkiem, kas palīdz uzturēt koda higiēnu automātiski.

Izpildes soļi:

  1. VS Code paplašinājumu veikalā (Extensions) atrodi un instalē "Black Formatter" vai "autopep8".
  2. Saraksti tīši nepareizi noformētu kodu (piem., a=1+2 bez atstarpēm).
  3. Izmanto taustiņu kombināciju Shift + Alt + F (Format Document).
  4. Novēro, kā dators pats sakārto atstarpes un tukšās rindas atbilstoši standartam.

3. uzdevums (Padziļināts): CYOA koda revīzija

Sagatavot iepriekšējās stundās rakstīto spēles prototipu lielajam noslēguma darbam.

Izpildes soļi:

  1. Atver savas CYOA spēles mapi un visus tajā esošos moduļus.
  2. Veic koda "auditu": vai visi import ir augšā? Vai konstantes ir ar LIELAJIEM burtiem?
  3. Katras funkcijas sākumā pievieno docstring (aprakstu trīskāršās pēdiņās """), paskaidrojot, ko funkcija dara.
  4. Pārliecinies, ka rindu garums nepārsniedz saprātīgas robežas (ap 79-88 rakstzīmes).

Jaunā viela: PEP 8 Galvenie Standarti

PEP 8 (Python Enhancement Proposal 8) ir dokuments, kas apraksta, kā jāizskatās Python kodam. To ievērošana ir obligāta prasme profesionālā vidē.

Kāpēc tas ir svarīgi?

Kodu daudz biežāk lasa, nekā raksta. Labi noformēts kods ietaupa laiku gan Tev, gan Taviem komandas biedriem. Atceries: Skaists kods ir strādājošs kods!
