Tavs šīs stundas izaicinājums: Nodalīt stāstu no koda. Tu iemācīsies izmantot Google Sheets, lai izveidotu strukturētu spēles pasauli, un eksportēt to CSV formātā. Tas ļaus Tev mainīt visu spēles stāstu, neaiztiekot nevienu Python koda rindiņu!

SR 2.3.5. (Datu strukturēšana izklājlapās)

1. uzdevums (Vienkāršs): Tabulas arhitekts

Izveidot precīzu struktūru Google Sheets, lai dators prastu "nolasīt" Tavu stāstu.

Izpildes soļi:

  1. Atver Google Sheets un izveido jaunu tabulu.
  2. Pirmajā rindā ieraksti galvenes: id, nosaukums, teksts, izvele_a, merkis_a, izvele_b, merkis_b.
  3. Izveido vismaz 3 savstarpēji saistītas lokācijas (piemēram, Sākums ved uz Mežu vai Alu).
  4. Svarīgi: merkis_a un merkis_b ailēs ieraksti tās lokācijas id, uz kuru spēlētājam jādodas.

2. uzdevums (Viduvējs): Pasaules eksports un CSV analīze

Pārvērst vizuālo tabulu mašīnlasāmā formātā un saprast, kā dati tiek atdalīti ar komatiem.

Izpildes soļi:

  1. Papildini savu tabulu, līdz Tev ir vismaz 8 lokācijas (iekļaujot uzvaras un zaudējumu beigas).
  2. Dodies uz File -> Download -> Comma Separated Values (.csv).
  3. Lejupielādēto failu pārsauc par pasaule.csv un ieliec sava projekta mapē.
  4. Atver failu ar VS Code – redzēsi, ka katra rinda ir atdalīta ar komatiem.

3. uzdevums (Padziļināts): Python "Pirmā lasīšana"

Pārbaudīt, vai Tavs Python dzinējs spēj piekļūt datiem, kas sagatavoti Google Sheets.

Izpildes soļi:

  1. VS Code izveido failu tests.py.
  2. Importē csv moduli.
  3. Atver pasaule.csv un, izmantojot csv.DictReader, izprintē visas lokācijas un to aprakstus.
  4. Pārliecinies, ka visi dati terminālī izskatās pareizi un nav dīvainu simbolu.

ebSkola Life Hacks:

1. ID ir svēts: Ja Tavā izvēlē ir mērķis 5, bet tabulā nav rindas ar ID 5, spēle "salūzīs".
2. Galvenes bez kļūdām: Python meklēs precīzi tādus vārdus, kādus Tu ierakstīji tabulas pirmajā rindā. Lieto mazos burtus un izvairies no atstarpēm.
3. Komati tekstā: Ja stāstā lieto komatus, Google Sheets tos automātiski ieliks pēdiņās. Python csv modulis ir pietiekami gudrs, lai ar to tiktu galā! 

import csv

with open('pasaule.csv', mode='r', encoding='utf-8') as f:
    # DictReader pārvērš katru rindu par vārdnīcu (dict)
    reader = csv.DictReader(f)
    
    for rinda in reader:
        # Piekļūstam datiem, izmantojot galveņu nosaukumus
        print(f"ID: {rinda['id']} | Vieta: {rinda['nosaukums']}")
