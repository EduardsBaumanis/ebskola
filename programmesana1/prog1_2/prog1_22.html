70 min darba sadalījums: 1. uzdevums (~20 min) — atjauno teorijas piemēru vai minimālo prototipu; 2. uzdevums (~25 min) — pielieto to galvenajā uzdevumā; 3. uzdevums (~25 min) — pārbaudi rezultātu, izlabo kļūdas un pieraksti, kas strādā. Teorija: Ievade un datu tipu tulkošana

Lai programma reaģētu uz spēlētāja darbībām, tai jāspēj pieņemt un pareizi interpretēt informācija no ārpasaules.

  • Input (Ievade): Funkcija input() aptur kodu un gaida lietotāja tekstu. Svarīgi atcerēties, ka visi dati no input() sākotnēji ir String (Simbolu virkne).
  • String (Simbolu virkne): Teksta datu tips str. Piemēram, "15" datoram ir tikai divi simboli, nevis skaitlis piecpadsmit.
  • Type casting (Datu tipu konvertācija): Funkciju izmantošana, lai mainītu datu tipu.
  • Integer (Vesels skaitlis): Funkcija int() pārvērš tekstu par veselu skaitli (piemēram, HP punkti, inventāra priekšmetu skaits).
  • Float (Daļskaitlis): Funkcija float() pārvērš tekstu par skaitli ar decimāldaļu (piemēram, 1.5 uzbrukuma reizinātājs).

  3. Pieraksti, kurā gala projekta daļā šī prasme būs vajadzīga: sākuma ekrānā, spēles laukumā, datos, vadībā, rezultātā vai publicēšanā.

Biežākās kļūdas (un kā tās labot):

  • ValueError (Vērtības kļūda): Rodas, ja int() vai float() funkcijā tiek nodots teksts, kas nav cipars (piemēram, "pieci"). Terminālī jāievada tikai matemātiskas vērtības.
  • TypeError (Tipa kļūda): Rodas, ja mēģina veikt matemātiskas darbības starp String un Integer. Pārliecinies, ka visi dati aprēķinos ir konvertēti uz skaitļiem.
  • Decimālkomats pret decimālpunktu: Python un terminālis Float vērtībās kā atdalītāju atpazīst tikai punktu (.), nevis komatu (,). Ievadot komatu, programma sabruks.

Koda piemērs: Datu iegūšana un konvertācija

# Teksta iegūšana
vards = input("Ievadi vārdu: ")

# Vesela skaitļa iegūšana
baze = int(input("Ievadi sākuma punktus: "))

# Daļskaitļa iegūšana
reizinatajs = float(input("Ievadi reizinātāju (piem., 1.2): "))

# Matemātika
gala_punkti = baze * reizinatajs

# Izvade
print("Dalībnieks", vards, "iegūst", gala_punkti, "punktus!")
Ievadi vārdu: Ozols
Ievadi sākuma punktus: 50
Ievadi reizinātāju (piem., 1.2): 1.5
Dalībnieks Ozols iegūst 75.0 punktus!
