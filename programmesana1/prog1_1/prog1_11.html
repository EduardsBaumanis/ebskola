1.1 Datora sagatavošana un IDE

Tavs šīs stundas izaicinājums: Sagatavot datoru profesionālam programmētāja darbam, uzstādīt atvērtā pirmkoda rīkus un palaist savu pirmo kodu Python valodā.

2.4.12. Sagatavo un konfigurē programmēšanas vidi.
3.1.1. Ievero ergonomikas un darba drošības principus.

Teorija: Kas ir IDE un Python?

Lai rakstītu kodu, nepietiek ar parastu teksta redaktoru. Mums ir nepieciešama specializēta vide.

  • Python: Programmēšanas valoda jeb "tulks", kas paskaidro datoram Tavas instrukcijas.
  • IDE (Integrated Development Environment): Programma koda rakstīšanai. Mēs izmantosim Visual Studio Code (VS Code) — tas ir bezmaksas, viegls un profesionāls rīks.
  • Extensions (Paplašinājumi): VS Code papildrīki. Lai VS Code "saprastu" Python, mums būs jāuzstāda atbilstošs spraudnis.
  • Ergonomika: Programmējot mugurai jābūt taisnai, ekrānam acu līmenī un ik pēc 20 minūtēm jāatpūtina redze (20-20-20 noteikums).
# Pirmā Python programma (hello.py)
print("Sveika, pasaule!")
# Palaiž ar Run pogu vai terminālī:
# python hello.py

1. uzdevums: Programmatūras uzstādīšana

Sagatavot datora bāzi, uzstādot Python interpretatoru un VS Code redaktoru.

Izpildes soļi:

  1. Atver Microsoft Store savā datorā.
  2. Meklētājā ieraksti "Python 3.12" un nospied "Get" vai "Install".
  3. Meklētājā ieraksti "Visual Studio Code" un uzstādi to.
  4. Atver uzstādīto VS Code programmu.

2. uzdevums: IDE konfigurēšana

Pielāgot VS Code vidi, lai tā atpazītu Python kodu un palīdzētu Tev to rakstīt.

Izpildes soļi:

  1. VS Code kreisajā vertikālajā joslā uzklikšķini uz Extensions ikonas (izskatās kā četri kvadrātiņi).
  2. Meklēšanas logā ieraksti "Python".
  3. Izvēlies pirmo rezultātu (autors: Microsoft) un nospied zilto pogu Install.
  4. Atver iestatījumus (Ctrl + ,) un meklē "Auto Save". Iestati to uz afterDelay, lai Tavs darbs saglabātos automātiski.

3. uzdevums: Pirmais "Sveiciens" pasaulei

Izveidot savu pirmo koda failu un pārliecināties, ka vide darbojas pareizi.

Izpildes soļi:

  1. VS Code izvēlies File -> New Text File un saglabā to (Ctrl + S) uz darbavirsmas ar nosaukumu hello.py.
  2. Ieraksti failā: print("Sveika, pasaule!").
  3. Nospied Run Python File (trīsstūra ikona loga augšējā labajā stūrī).
  4. Apskati rezultātu Terminal (Terminālī) loga apakšā.
  5. Iesūtīt uzdevuma github saiti e-klasē. (Piezīme: Ja vēl neesi izveidojis GitHub, šoreiz vari izlaist vai sekot skolotāja norādēm).

Papildus uzdevums: ASCII māksla terminālī

Eksperimentē ar teksta izvadi, lai terminālī izveidotu vizuālu attēlu.

Izpildes soļi:

  1. Izveido jaunu failu art.py.
  2. Izmanto vairākas print() funkcijas pēc kārtas.
  3. Izmantojot burtus un simbolus (piem., / \ | _ *), uzzīmēt robota vai mājas siluetu.
  4. Palaid kodu un pārbaudi, vai simboli terminālī nav "nobīdījušies".

Biežākās kļūdas (un kā tās labot):

  • "Python is not recognized": Restartē VS Code programmu pēc Python instalācijas, lai tā pamanītu jauno "tulku".
  • SyntaxError: Esi aizmirsis pēdiņas vai iekavas print() funkcijai. Pārbaudi piemēru koda sadaļā!
  • Balts aplītis pie faila nosaukuma: Fails nav saglabāts. Lieto Ctrl+S vai ieslēdz Auto Save.
