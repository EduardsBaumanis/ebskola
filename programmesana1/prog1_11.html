1. tēma: Ievads programmēšanā — izstrādes vide, komandrinda un projekta struktūra

Stundas sasniedzamais rezultāts

Stundas beigās skolēns izskaidro izstrādes vides nozīmi, uzstāda Python un IDE, izveido projekta mapes struktūru un izpilda programmu “Hello, World!” gan IDE, gan komandrindā. (SR: 2.4.12; 2.3.9; 2.3.11; 3.1.1)

1. uzdevums — Python un IDE uzstādīšana + pārbaude

Attēla avots: allthings.how (Python instalatora logs ar izvēli “Add python.exe to PATH”).

Mērķis: lai Python var palaist gan no IDE, gan no komandrindas.

A. Uzstādi Python
1) Lejupielādē Python instalatoru no python.org.
2) Windows vidē instalācijas pirmajā logā atzīmē izvēli “Add python.exe to PATH” (skat. attēlu augstāk).
3) Spied Install Now (vai Customize installation, ja skolotājs norāda citādi) un pabeidz instalāciju.

B. Uzstādi IDE (ieteicams VS Code)
1) Lejupielādē un uzstādi VS Code.
2) Atver VS Code → kreisajā izvēlnē Extensions (paplašinājumi).
3) Meklē “Python” un uzstādi Python paplašinājumu (Microsoft).

C. Pārbaudi Python komandrindā
1) Atver komandrindu/termināli:
  • Windows: Start → ieraksti cmd vai PowerShell
  • macOS: Applications → Utilities → Terminal
2) Ievadi komandu un spied Enter:
python --version
(Ja Windows vidē nestrādā, mēģini: py --version)
3) Iegūto Python versiju pieraksti failā README.md (sadaļā “Mana darba vide”).

Ja redzi kļūdu (piem., “python is not recognized…”): pieraksti kļūdas tekstu un atzīmē, ko mēģināji darīt.

2. uzdevums — projekta mape un “Hello, World!” palaišana

Attēla avots: code.visualstudio.com (VS Code panelis ar “TERMINAL” cilni).

Mērķis: izveidot korektu projekta struktūru un palaist programmu divos veidos.

A. Izveido projekta mapi
1) Izvēlies vietu, kur glabāsi programmēšanas darbus (ieteicams Documents vai mape Projekti).
2) Izveido mapi ar nosaukumu: projekts_01

B. Atver projektu VS Code
1) VS Code izvēlnē: File → Open Folder…
2) Izvēlies mapi projekts_01 un apstiprini.

C. Izveido failus
1) VS Code kreisajā pusē (Explorer) izveido failu main.py.
2) Turpat izveido failu README.md.

D. Uzraksti un palaid programmu IDE
1) Atver main.py un ieraksti kodu (skat. zemāk).
2) Palaid programmu ar Run (vai “Run Python File”).
3) Pārliecinies, ka redzi tekstu “Hello, World!”.

E. Uzraksti un palaid programmu VS Code terminālī
1) Atver termināli VS Code: View → Terminal (skat. attēlu augstāk).
2) Pārliecinies, ka termināļa ceļš ir uz projekta mapi (ja nē — atver termināli no projekta mapes vai izmanto cd).
3) Ievadi:
python main.py
(vai Windows vidē: py main.py)
4) Ja redzi “Hello, World!”, uzdevums izpildīts. 


print("Hello, World!")

3. uzdevums — README aizpildīšana un darba vides apraksts

Attēla avots: docs.github.com (README.md attēlojums repozitorijā).

Mērķis: izveidot īsu, saprotamu projekta aprakstu, lai cits cilvēks varētu to palaist bez minēšanas.

A. Atver README failu
1) VS Code kreisajā pusē (Explorer) atver README.md.
2) Ievieto zemāk redzamo sagatavi un aizpildi iekavas ar savu informāciju.

B. Pārbaude (paškontrole)
Pirms iesniegšanas pārbaudi:
• vai README ir Python versija (no 1. uzdevuma),
• vai ir īss IDE pamatojums (1–2 teikumi),
• vai ir instrukcija, kā palaist programmu (komanda).

C. Saglabā izmaiņas
Nospied Ctrl+S (Windows) vai Cmd+S (macOS).

Papildus informācija

PATH izvēle instalācijā (1. uzdevums) nozīmē, ka sistēma “atrod” Python no jebkuras mapes. Tāpēc komanda python --version strādā uzreiz, un vēlāk vari palaist projektus, neatverot tos vienmēr no vienas konkrētas vietas.

Atvērt mapi (nevis vienu failu) (2. uzdevums) ir svarīgi, jo VS Code tad uztver projektu kā vienotu darba telpu: redzi visus failus, vari izmantot iebūvēto termināli un vieglāk saproti, kur tieši atrodas main.py.

README (3. uzdevums) ir projekta “īsā instrukcija”. Ja tajā ir “ko dara”, “kā palaist” un “kāda vide”, tad darbu var atkārtot arī pēc mēneša, un skolotājam ir skaidrs, ka uzdevums izpildīts apzināti, nevis nejauši.
