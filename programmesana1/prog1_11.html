Tēmas Sasniedzamie rezultāti Palīgmateriāli

1. tēma: Ievads un darba vide. 1. stunda: Datora sagatavošana darbam.

Tavs šīs stundas izaicinājums: Tu iemācīsies atpazīt programmēšanas vides galvenos rīkus, patstāvīgi sagatavot savu datoru profesionālam darbam un palaist savu pirmo kodu, rūpējoties par savu veselību un drošību.

SR 2.4.12. (Vide), 2.3.11. (Iestatījumi efektivitātei), 3.1.1. (Ergonomika)

Teorija: Profesionāla darba vide

  • IDE (Visual Studio Code): Tavs galvenais darbagalds. Tā nav tikai rakstāmmašīna, bet gan vieda vide, kas palīdz krāsot kodu, rādīt kļūdas un pārvaldīt failus.
  • Python interpretators: Programma jeb "dzinējs", kas nolasa Tavu rakstīto kodu un pārvērš to datoram saprotamās instrukcijās.
  • Paplašinājumi (Extensions): VS Code papildrīki (spraudņi). Mums būs nepieciešams "Python" paplašinājums, lai IDE saprastu šo programmēšanas valodu.
  • Ergonomika un drošība: Darbs pie datora prasa rūpes par veselību. Mugurai jābūt taisnai, ekrānam acu līmenī un ik pēc 20 minūtēm jāatpūtina redze (20-20-20 noteikums).

1. uzdevums : Uzstādīšana un pirmais tests

Sagatavot datoru darbam un pārliecināties, ka tas spēj izpildīt Python kodu.

Izpildes soļi:

  1. Windows uzdevumu joslā atver Microsoft Store.
  2. Meklēšanā ieraksti "Python 3.12" un nospied "Get" vai "Install".
  3. Uzstādi arī "Visual Studio Code" no tā paša veikala vai oficiālās vietnes.
  4. Atver VS Code un izveido jaunu failu: File -> New Text File.
  5. Ieraksti testa kodu: print("Sveika, pasaule!").
  6. Saglabā failu (Ctrl+S) uz Desktop ar nosaukumu tests.py.
  7. Nospied Run Python File (trīsstūra poga loga augšējā labajā stūrī) un vēro rezultātu Terminal logā.

2. uzdevums : Projekta mapes izveide un rīku iestatīšana

Sakārtot darba vidi topošajai spēlei un pievienot palīgrīkus efektivitātes paaugstināšanai.

Izpildes soļi:

  1. Uz Desktop izveido jaunu mapi mana_spele.
  2. VS Code izvēlies File -> Open Folder un atver izveidoto mapi.
  3. Kreisajā vertikālajā joslā uzklikšķini uz Extensions ikonas (četri kvadrātiņi).
  4. Meklē "Python" (izstrādātājs Microsoft) un instalē to.
  5. Izveido mapē jaunu failu sveiciens.py un ieraksti kodu, kas izvada tekstu par gatavību izstrādei.

3. uzdevums : Efektivitātes iestatījumi un plānošana

Uzsākt projekta dokumentēšanu un pielāgot darba vidi personīgajām vajadzībām.

Izpildes soļi:

  1. Savā darba mapē izveido plānošanas failu planosana.txt.
  2. Ieraksti tajā savas spēles ieceri: žanru, galveno varoni un mērķi.
  3. Atver VS Code iestatījumus (Ctrl + ,) un meklētājā ieraksti "Breadcrumbs". Pārliecinies, ka tie ir ieslēgti (palīdz navigācijai).
  4. Iestatījumos atrodi "Editor: Font Size" un iestati sev ērtāko izmēru (ieteicams 14 vai 16), lai saudzētu redzi.
  5. Pārbaudi, vai visi faili ir redzami Explorer (Ctrl+Shift+E) sadaļā un ir saglabāti.

Biežākās kļūdas (un kā tās labot):

  • Balts aplītis pie faila nosaukuma: Tas nozīmē, ka izmaiņas nav saglabātas. Lieto Ctrl+S!
  • Kļūda "Python was not found": Ja Python tikko instalēts, restartē VS Code programmu.
  • Fails neatveras kā kods: Pārbaudi paplašinājumu. Tam jābūt .py, nevis .txt vai .py.txt.

Zini, kāpēc tas ir svarīgi? Programmēšana nav tikai koda rakstīšana – tā ir prasme pārvaldīt savus rīkus. Sakārtota vide ļauj Tev veltīt visu enerģiju spēles loģikas radīšanai, nevis cīņai ar tehniku.
