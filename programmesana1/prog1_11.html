Palīgmateriāli

1. tēma: Ievads un darba vide. 1. stunda: Datora sagatavošana darbam.

Tavs šīs stundas izaicinājums: Tu iemācīsies atpazīt programmēšanas vides galvenos rīkus, patstāvīgi sagatavot savu datoru profesionālam darbam un palaist savu pirmo kodu, rūpējoties par savu veselību un drošību.

SR 2.4.12. (Vide), 2.3.11. (Iestatījumi efektivitātei), 3.1.1. (Ergonomika)

1. uzdevums (Vienkāršs): Uzstādīšana un pirmais tests

Sagatavot datoru darbam un pārliecināties, ka tas saprot Python valodu.

Izpildes soļi:

  1. Windows uzdevumu joslā (ekrāna apakšā) atrodi un atver Microsoft Store lietotni.
  2. Meklēšanas logā ieraksti "Python 3.12" un nospied pogu "Iegūt" vai "Instalēt".
  3. Tajā pašā veikalā meklē lietotni "Visual Studio Code" un uzstādi arī to.
  4. Atver Visual Studio Code un augšējā izvēlnē spied File -> New Text File.
  5. Ieraksti kodu: print("Sveika, pasaule!").
  6. Saglabā failu (izmanto taustiņu kombināciju Ctrl+S) uz darbvirsmas (Desktop) ar nosaukumu tests.py.
  7. Nospied mazo trīsstūra pogu (Run) loga augšējā labajā stūrī un vēro rezultātu loga apakšā.

2. uzdevums (Viduvējs): Projekta mapes izveide un rīku iestatīšana

Sakārtot darba vidi topošajai spēlei un pievienot palīgrīkus ērtākam darbam.

Izpildes soļi:

  1. Uz datora darbvirsmas (Desktop) izveido jaunu mapi ar nosaukumu mana_spele.
  2. VS Code programmā spied File -> Open Folder un izvēlies savu izveidoto mapi.
  3. VS Code kreisajā sānā uzmeklē pogu Extensions (ikona ar četriem kvadrātiņiem).
  4. Meklētājā ieraksti "Python" un uzinstalē pirmo rezultātu, kura izstrādātājs ir Microsoft.
  5. Izveido savā mapē jaunu failu ar nosaukumu sveiciens.py un ieraksti kodu: print("Šī būs mana pirmā spēle!"). Saglabā to.

3. uzdevums (Padziļināts): Spēles idejas dokumentēšana

Uzsākt projekta plānošanu un apgūt navigāciju programmā.

Izpildes soļi:

  1. Savā projekta mapē izveido jaunu failu ar nosaukumu apraksts.txt.
  2. Šajā failā ieraksti atbildes uz trim jautājumiem: kāds būs tavas spēles nosaukums, kur norisināsies darbība un kas būs galvenais varonis.
  3. VS Code iestatījumos (zobratiņa ikona -> Settings) ieraksti meklētājā vārdu "Breadcrumbs".
  4. Pārliecinies, ka pie Breadcrumbs ir ielikts ķeksītis (tas palīdzēs loga augšā vienmēr redzēt, kurā mapē atrodies).
  5. Pārbaudi, vai visi faili (sveiciens.py un apraksts.txt) ir pareizi saglabāti un redzami kreisās puses sarakstā.

Jaunās vielas kopsavilkums

  • Microsoft Store loma: Skolas datoros tas ir drošākais veids, kā ielādēt pārbaudītas programmas.
  • IDE (VS Code) pret Python: VS Code ir tavs darbagalds (vieta, kur raksti), bet Python ir dzinējs, kas šo tekstu izpilda.
  • Paplašinājumi: Papildrīki, kas palīdz programmētājam pamanīt kļūdas uzreiz un rakstīt kodu ātrāk.
  • Ergonomika: Rūpējies par veselību — sēdi taisni, ekrāns acu līmenī un ievēro "20-20-20" noteikumu acu atpūtai.

Biežākās kļūdas (un kā tās labot):

  • Kods nedarbojas, lai gan viss uzrakstīts pareizi: Pārliecinies, vai esi saglabājis failu (Ctrl+S). Ja faila cilnes augšā redzi baltu aplīti, tas nozīmē, ka izmaiņas nav saglabātas.
  • VS Code nepiedāvā palaist Python kodu (nav trīsstūra pogas): Pārbaudi, vai faila nosaukuma beigās ir .py paplašinājums. Bez tā dators nezinās, ka tas ir koda fails.
  • Kļūda "Python was not found": Ja tikko uzinstalēji Python, aizver VS Code un atver to vēlreiz. Windows sistēmai ir jāpārlādējas, lai "redzētu" jauno programmu.
  • Nevaru atrast savu failu kreisajā sarakstā: Atceries, ka VS Code vislabāk strādā, ja atver mapi (File -> Open Folder), nevis tikai atsevišķu failu.
  • Teksta fails (apraksts.txt) neizpildās kā kods: Tas ir pareizi! .txt faili ir tikai informācijai, tos nevar "palaist" kā programmu.
