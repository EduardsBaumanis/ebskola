3. tēma: Vadības struktūras. 1. stunda: Salīdzināšanas operatori un if/elif/else.

Tavs šīs stundas izaicinājums: Tu iemācīsi datoram pieņemt lēmumus. No lineāra koda mēs pārejam pie zarošanās loģikas, kur programma pati izvēlas ceļu, balstoties uz situāciju.

Praktiskie uzdevumi un darbību secība

1. uzdevums: Izvēles pārbaude (if/else)

  1. VS Code izveido failu parbaude.py.
  2. Izveido mainīgo izvele un piešķir tam vērtību no input("Izvēlies (akmens/skeres/papirs): ").
  3. Uzraksti if rindu, kas pārbauda: if izvele == "akmens": (neaizmirsti divas vienādības zīmes!).
  4. Nākamajā rindā ar atkāpi (Tab) ieraksti: print("Akmens ir stabila izvēle!").
  5. Atgriezies sākuma pozīcijā (bez atkāpes) un uzraksti else:.
  6. Zem else ar atkāpi ieraksti: print("Tu izvēlējies kaut ko citu.").
  7. Palaid programmu un izmēģini abus variantus!

2. uzdevums: Tiesnesis V1 (if/elif/else)

  1. Izveido failu tiesnesis_v1.py.
  2. Definē datora izvēli: dators = "akmens".
  3. Iegūsti lietotāja ievadi mainīgajā cilveks.
  4. Pārbaudi neizšķirtu: if cilveks == dators: -> izvadi "Neizšķirts!".
  5. Pārbaudi zaudējumu: elif cilveks == "skeres": -> izvadi "Zaudējums! Akmens salauž šķēres.".
  6. Pārbaudi uzvaru: elif cilveks == "papirs": -> izvadi "Uzvara! Papīrs ietin akmeni.".
  7. Pievieno else: kļūdainiem vārdiem -> izvadi "Nezināms gājiens!".

3. uzdevums: Skaitļu duelis (Matemātiskā loģika)

  1. Izveido failu skaitlu_duelis.py.
  2. Iegūsti divus skaitļus sk1 un sk2, obligāti konvertējot tos uz int().
  3. Salīdzini: if sk1 > sk2:. Izmantojot f-string, paziņo: "Pirmais ir lielāks par {sk1 - sk2}".
  4. Pievieno elif sk1 == sk2: paziņojumam par neizšķirtu.
  5. Pievieno else: (kas šajā gadījumā nozīmē, ka $sk2 > sk1$) un paziņo otrā skaitļa uzvaru.

1. Daļa: Salīdzināšanas operatori

Operators Nozīme Piemērs
== Vienāds ar "A" == "a" (False)
!= Nav vienāds ar 5 != 10 (True)
> / < Lielāks / Mazāks 10 > 2 (True)

2. Daļa: Zarošanās struktūra

Elements Loma Cik reizes?
if Sākuma pārbaude. Vienu reizi.
elif Papildu pārbaudes (ja if bija aplams). No 0 līdz bezgalībai.
else Izpildās, ja nekas cits "nenostrādāja". 0 vai 1 reizi beigās.

Datora prioritātes:

Dators lasa nosacījumus kā rindas kārtībā. Tiklīdz tas atrod pirmo True, tas izpilda kodu un uzreiz lec ārā no visas if/elif/else struktūras. Tas nepārbauda nākamos variantus, ja viens jau ir derējis!


# Atceries par kolu un atkāpi!
vecums = 18

if vecums >= 18:
    print("Vari balsot.") # Šī rinda ir ar atkāpi!
else:
    print("Vēl par jaunu.")
