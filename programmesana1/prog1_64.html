SR 2.4.14. (Datu struktūru kārtošana un JSON)

1. uzdevums (Vienkāršs): Pirmais ieraksts

Sagatavot JSON struktūru un saglabāt pirmo spēlētāju kā vārdnīcu sarakstā.

Izpildes soļi:

  1. VS Code izveido failu rezultati.py un importē json.
  2. Izveido tukšu sarakstu visi_rezultati = [].
  3. Pajautā lietotājam vārdu un punktus.
  4. Izveido vārdnīcu dati = {"vards": vards, "punkti": int(punkti)}.
  5. Pievieno vārdnīcu sarakstam un saglabā to failā leaderboard.json.

2. uzdevums (Viduvējs): Hall of Fame papildinātājs

Iemācīties nolasīt esošos rezultātus, pirms pievienot jaunu, lai dati neizdzēstos.

Izpildes soļi:

  1. Pirms jauna rezultāta prasīšanas, mēģini nolasīt leaderboard.json.
  2. Izmanto try...except FileNotFoundError, lai, ja fails neeksistē, visi_rezultati būtu tukšs saraksts.
  3. Pievieno jauno rezultātu sarakstam un saglabā atpakaļ failā.
  4. Palaid programmu vairākas reizes un pārbaudi, vai failā punkti uzkrājas.

3. uzdevums (Padziļināts): TOP 5 kārtošanas sistēma

Iemācīties sakārtot sarakstu pēc vārdnīcas atslēgas un saglabāt tikai labākos rezultātus.

Izpildes soļi:

  1. Pēc jauna rezultāta pievienošanas sakārto sarakstu no lielākā uz mazāko.
  2. Izmanto: saraksts.sort(key=lambda x: x['punkti'], reverse=True).
  3. Izmanto slicing (nogriešanu), lai saglabātu tikai pirmos piecus: saraksts[:5].
  4. Izveido funkciju paradit_top(), kas izvada rezultātus glītā tabulā.

Teorija: JSON apstrādes cikls

JSON faili nav kā parasts teksts, kuram var "pierakstīt klāt" rindas beigās. Tie ir vienoti datu objekti. Lai tos papildinātu, jāievēro trīs soļi:

  1. LASĪT: Ielādē visu failu Python sarakstā.
  2. MAINĪT: Pievieno jaunu elementu un sakārto sarakstu.
  3. RAKSTĪT: Pārraksti visu failu ar jauno, papildināto sarakstu.

Kārtošana ar Lambda

Kad sarakstā ir vārdnīcas, Python nezin, pēc kura "stabiņa" kārtot. Mums ir jādod "ceļa rādītājs":

$$key=lambda\ x:\ x['punkti']$$

Tas pasaka: "Skaties uz katru elementu x un kārtošanai izmanto tā vērtību pie atslēgas 'punkti'."

Biežākās kļūdas:

1. Pārrakstīšana: Ja Tu aizmirsti vispirms nolasīt esošos datus, katrs jauns spēlētājs izdzēsīs iepriekšējo.
2. Datu tipi: Ja punkti tiks saglabāti kā teksts "9", kārtojot tas būs "lielāks" par "100". Vienmēr lieto int()!
3. indent=4: Saglabājot JSON, lieto indent=4 parametru, lai fails būtu lasāms cilvēkam.

Piemērs: Rezultātu ielāde un kārtošana


import json

# 1. Mēģinām nolasīt datus
try:
    with open("scores.json", "r") as f:
        top_saraksts = json.load(f)
except FileNotFoundError:
    top_saraksts = []

# 2. Pievienojam jaunu rezultātu
jauns = {"vards": "Drosmīgais", "punkti": 150}
top_saraksts.append(jauns)

# 3. Sakārtojam un saglabājam TOP 5
top_saraksts.sort(key=lambda x: x['punkti'], reverse=True)
top_5 = top_saraksts[:5]

with open("scores.json", "w") as f:
    json.dump(top_5, f, indent=4)
