2. Tēma — Stils, multimediji un datņu pārvaldība

Par šo tēmu

Ja HTML ir spēles skelets, tad šajā tēmā mēs tai piešķirsim "ādu" un "balsi". Tu apgūsi CSS (Cascading Style Sheets), lai kontrolētu spēles vizuālo tēlu — krāsas, fontus un izvietojumu, ievērojot ergonomikas principus.

Mēs strādāsim ar multimediju resursiem: mācīsimies apstrādāt attēlus un skaņas failus, lai tie būtu piemēroti tīmekļa videi. Tāpat Tu apgūsi datņu pārvaldības labo praksi, organizējot sarežģītu projektu mapēs, lai Tavs kods un resursi būtu viegli atrodami un sinhronizējami GitHub vidē.
