3.3. tēma: Loģiskie operatori un sazarojumi (If/Else)

Tavs šīs stundas izaicinājums: Analizēt spēles stāvokli un izstrādāt algoritma sazarojumus, lai programma spētu patstāvīgi pieņemt lēmumus un reaģēt uz dažādām spēlētāja situācijām.

2.6.1. Lasa un izpilda algoritmus ar sazarojumiem; 5.2.1. Kritiskā domāšana problēmu risināšanā.

Teorija: Lēmumu pieņemšana kodā

Lai spēle zinātu, kad spēlētājs ir uzvarējis vai zaudējis, mēs izmantojam nosacījumus (if statements). Tie pārbauda, vai apgalvojums ir patiess (true) vai aplams (false).

  • Salīdzināšana: > (lielāks), < (mazāks), === (precīzi vienāds), !== (nav vienāds). Noteikti ievēro, ka viens vienādības zīmes = nozīmē piešķirt vērtību, bet trīs === nozīmē salīdzināt!
  • Struktūra: if (nosacījums) { dari šo } else { dari citu }.
  • Loģiskie operatori: && (UN - abiem jābūt patiesiem), || (VAI - vismaz vienam jābūt patiesam). Piemēram: Ja punkti > 100 UN laiks > 0, tad uzvara.

1. uzdevums: Vienkāršs sazarojums (If)

Izveidot loģiku, kas pārbauda spēlētāja punktu skaitu un paziņo par sasniegumu.

Izpildes soļi:

  1. Savā script.js failā izveido mainīgo let punkti = prompt("Cik punktus tu ieguvi?"); un pārveido to par skaitli (izmanto Number()).
  2. Uzraksti if bloku: pārbaudi, vai punkti >= 100 (lielāks vai vienāds ar 100).
  3. Ja nosacījums izpildās, izvadi konsolē vai ar alert() ziņojumu: "Apsveicu, tu esi atbloķējis slepeno līmeni!". Pārbaudi pārlūkā, ievadot 150 un pēc tam 50.

2. uzdevums: Alternatīvais ceļš (If / Else)

Papildināt algoritmu ar rīcību gadījumam, ja nosacījums netiek izpildīts (piemēram, "Game Over").

Izpildes soļi:

  1. Pievieno iepriekšējam if blokam else sadaļu.
  2. Iekš else bloka figūriekavām {} ieraksti ziņojumu: "Diemžēl tev nepietiek punktu. Mēģini vēlreiz!".
  3. Izveido vēl vienu mainīgo let dzivibas = 0; un ar jaunu if/else pārbaudi: ja dzīvības ir lielākas par 0, izvadi "Spēle turpinās", citādi izvadi "Game Over".

3. uzdevums: Sarežģīta loģika (Else If un AND/OR)

Izmantot vairākus nosacījumus vienā blokā, lai spēle reaģētu atšķirīgi uz dažādiem punktu diapazoniem.

Izpildes soļi:

  1. Izveido punktu rangu sistēmu. Sāc ar if (punkti > 1000) { ... "Tu esi meistars!" }.
  2. Pievieno else if (punkti > 500 && punkti <= 1000) { ... "Labs rezultāts!" }.
  3. Pievieno beigās else { ... "Tev vēl jātrenējas!" }. Pārbaudi kodu, ievadot dažādus skaitļus, un sinhronizē failus ar GitHub.

Biežākās kļūdas (un kā tās labot):

  • Viens vienādības zīmes = izmantošana if iekavās: if (dzivibas = 0) vienmēr būs aplams un nejauši nomainīs tavas dzīvības uz 0. Vienmēr izmanto === salīdzināšanai!
  • Aizmirstas figūriekavas {}: Ja aizmirsīsi figūriekavas, JavaScript izpildīs tikai pirmo rindu pēc if, kas var novest pie haotiskas programmas darbības.
  • Nepareiza secība else if blokos: Vienmēr sāc pārbaudīt no visstingrākā vai lielākā nosacījuma uz mazāko. Ja sāksi ar punkti > 0, tad kods nekad nenonāks līdz punkti > 1000 pārbaudei.

Koda piemērs

let spelesPunkti = 850;
let irAtslega = true;

// 1. Vienkāršs If/Else
if (spelesPunkti >= 1000) {
    console.log("Uzvara! Izcils sniegums.");
} else {
    console.log("Spēle turpinās. Krāj punktus!");
}

// 2. Else If un Loģiskais operators (&&)
if (spelesPunkti >= 500 && irAtslega === true) {
    document.getElementById("rezultats").innerText = "Tu atvērsi zelta lādi!";
} else if (spelesPunkti >= 500 && irAtslega === false) {
    document.getElementById("rezultats").innerText = "Tev vajag atslēgu!";
} else {
    document.getElementById("rezultats").innerText = "Bīstama teritorija...";
}
Konsoles izvade: Spēle turpinās. Krāj punktus!
HTML elementa izvade: Tu atvērsi zelta lādi!
