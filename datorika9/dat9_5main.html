# 02 špikeris

Publicēšanas špikeris

Git darbplūsma, projekta pārbaudes saraksts un README veidne.

SolisKomanda / darbībaPiezīme
1. Pievienot failusgit add .Visus izmainītos failus stages
2. Commitgit commit -m "teksts"Saglabā izmaiņas ar aprakstu
3. Pushgit push origin mainAugšupielādē uz GitHub
4. GitHub PagesSettings → Pages → main/rootIeslēdz publicēšanu
5. URLlietotajs.github.io/spelePieejams ~1-2 minūtes pēc push

Gatava projekta pārbaudes saraksts

  • index.html - galvenā lapa (GitHub Pages to atver pirmā)
  • style.css - visi stili strādā ar relatīvām URL
  • game.js - spēles loģika darbojas bez konsolē kļūdām
  • README.md - projekta apraksts, ekrānuzņēmumi, vadības instrukcija
  • LICENSE - MIT vai cita brīvā licence
  • .gitignore - node_modules/, .DS_Store, *.env izslēgti
  • Spēle testēta divos pārlūkos (Chrome + Firefox)
  • DevTools konsolē nav neapstrādātu kļūdu
