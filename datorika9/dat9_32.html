3.2. tēma: Datu tipi un interaktivitāte (Input/Output)

Tavs šīs stundas izaicinājums: Saņemt datus no lietotāja, pareizi apstrādāt dažādus datu tipus un dinamiski izvadīt rezultātu HTML lapā.

2.6.1. Izmanto atbilstošas datu struktūras; 2.4.7. Veido kompleksu pieeju uzdevuma atrisināšanai.

Teorija: Kā dators saprot informāciju?

Lai gan cilvēkam "5" un 5 ir viens un tas pats, datoram tie ir atšķirīgi datu tipi:

  • Teksts (String): Jebkas, kas ielikts pēdiņās "Sveiki!" vai "5". Ar tekstu nevar veikt matemātiku.
  • Skaitlis (Number): Cipari bez pēdiņām, piemēram, 5 vai 3.14.
  • Loģiskā vērtība (Boolean): Var būt tikai true (patiess) vai false (aplams). Ideāli piemērots stāvokļiem, piemēram, irDzivs = true.
Interaktivitāte: Mēs varam prasīt lietotājam ievadīt datus ar prompt() un ievietot rezultātu tieši HTML lapā, izmantojot document.getElementById().innerText.

1. uzdevums: Lietotāja ievades saņemšana (Input)

Izmantot uznirstošo logu, lai pajautātu spēlētājam viņa vārdu un saglabātu to mainīgajā.

Izpildes soļi:

  1. Savā script.js failā izveido jaunu mainīgo: let speletajaVards = prompt("Kā tevi sauc, spēlētāj?");.
  2. Izmanto alert() funkciju, lai izvadītu sveicienu: alert("Sveicināts spēlē, " + speletajaVards + "!");.
  3. Palaid lapu pārlūkā un pārbaudi, vai uznirstošie logi parādās un pareizi izvada tavu ievadīto vārdu.

2. uzdevums: Matemātika un Datu tipu pārveidošana

Demonstrēt atšķirību starp teksta apvienošanu un reālu matemātisku saskaitīšanu.

Izpildes soļi:

  1. Paprasi lietotājam ievadīt sākuma bonusu: let bonuss = prompt("Ievadi sākuma punktus (1-10):");
  2. Saskaiti esošos punktus ar bonusu: let kopejiePunkti = 100 + bonuss; un izvadi to konsolē. Pamanīsi kļūdu (100 + "5" kļūs par "1005").
  3. Izlabo kļūdu, pārveidojot tekstu par skaitli: let kopejiePunkti = 100 + Number(bonuss);. Tagad matemātika strādās pareizi!

3. uzdevums: Dinamiska izvade HTML lapā (Output)

Aizstāt kaitinošos alert logus ar elegantu datu izvadi tieši spēles ekrānā.

Izpildes soļi:

  1. Savā index.html failā pievieno elementu ar unikālu ID, piemēram: <h2>Spēlētājs: <span id="varda-lauks">Nezināms</span></h2>.
  2. Sava script.js faila beigās ieraksti komandu, kas atrod šo elementu un nomaina tā tekstu: document.getElementById("varda-lauks").innerText = speletajaVards;.
  3. Veic Commit un Push uz GitHub ar ziņu "Pievienota spēlētāja ievade un DOM manipulācija".

Biežākās kļūdas (un kā tās labot):

  • "null" kļūda (Cannot set properties of null): Tas notiek, ja JS fails tiek ielādēts PIRMS HTML ir paspējis uzzīmēt tavu elementu ar attiecīgo ID. Pārliecinies, ka <script> tags HTML failā ir pašās beigās, tieši pirms </body>.
  • Teksta un mainīgo jaukšana: Atceries – teksts ir pēdiņās "vards", bet mainīgais ir bez pēdiņām vards.
  • Nepareizs lielais/mazais burts ID nosaukumā: getElementById("Vards") neatradīs id="vards". Jābūt 100% sakritībai.

Koda piemērs

// 1. Saņemam datus (Input) kā tekstu (String)
let spelesVards = prompt("Ievadi sava varoņa nosaukumu:");
let ievaditiePunkti = prompt("Cik punktus vēlies sākumā?");

// 2. Pārveidojam tekstu par skaitli un veicam aprēķinu
let saktSpeliAr = Number(ievaditiePunkti) + 50; 

// 3. Mainām HTML saturu (Output)
document.getElementById("varda-lauks").innerText = spelesVards;
document.getElementById("punktu-lauks").innerText = saktSpeliAr;
