3.1. tēma: Ievads JavaScript un mainīgie

Tavs šīs stundas izaicinājums: Izveidot ārējo JavaScript failu, savienot to ar HTML dokumentu un definēt mainīgos, lai saglabātu un manipulētu ar spēles pamatdatiem.

2.6.1. Pieraksta, lasa un izpilda algoritmus; 5.2.1. Kritiskā domāšana un problēmrisināšana.

Teorija: Smadzenes un Atmiņa (Mainīgie)

JavaScript (JS) ir programmēšanas valoda, kas padara lapu interaktīvu. Lai programma varētu kaut ko aprēķināt (piemēram, spēlētāja punktus), tai šī informācija ir kaut kur jāsaglabā. Mēs tam izmantojam mainīgos (kā kastes ar nosaukumu, kurās ielikt datus).

Datu uzglabāšanai mūsdienās izmanto divus galvenos atslēgas vārdus:

  • let mainiga_nosaukums = vērtība; – mainīgais, kura vērtība spēles laikā drīkst mainīties (piem., punkti, dzīvības).
  • const konstantes_nosaukums = vērtība; – konstante, kuras vērtība nekad nemainās (piem., maksimālais līmeņu skaits vai gravitācijas spēks).

Izstrādes laikā mēs bieži izmantojam console.log(), lai nosūtītu ziņojumus paši sev (programmētājiem) uz pārlūka slēpto konsoli.

1. uzdevums: Skripta faila pieslēgšana un Konsoles atvēršana

Izveidot JS failu un izvadīt paziņojumu pārlūka izstrādātāja rīkos (Developer Tools).

Izpildes soļi:

  1. Savā VS Code projekta mapē izveido jaunu failu script.js (turpat kur index.html).
  2. Failā index.html pirms aizverošā </body> taga, pievieno: <script src="script.js"></script>.
  3. Ieraksti script.js failā komandu: console.log("Spēle ir sākusies!!");
  4. Pārlūkā (Google Chrome) nospied F12 (vai labais klikšķis -> Inspect) un atver cilni Console. Tev tur jāredz savs ziņojums!

2. uzdevums: Spēles stāvokļa mainīgo izveide

Izmantojot let un const, definēt datus, kas būs nepieciešami tavas spēles darbībai.

Izpildes soļi:

  1. script.js definē trīs mainīgos ar let: spēlētāja vārdu, sākuma punktus (piem., 0) un dzīvības (piem., 3).
  2. Definē vienu konstanti ar const (piemēram, maksimālo punktu skaitu uzvarai).
  3. Izvadi šos datus konsolē, izmantojot teksta apvienošanu: console.log("Spēlētājs: " + spelesVards + " sāks ar " + dzivibas + " dzīvībām.");

3. uzdevums: Matemātiskās darbības ar mainīgajiem

Simulēt spēles notikumus, mainot izveidoto mainīgo vērtības un atjauninot rezultātu konsolē.

Izpildes soļi:

  1. Uzraksti kodu, kas simulē ienaidnieka uzbrukumu, samazinot dzīvību skaitu par 1 (piemēram, dzivibas = dzivibas - 1;).
  2. Uzraksti kodu, kas simulē monētas savākšanu, palielinot punktu skaitu par 10.
  3. Vēlreiz izvadi jaunos datus konsolē (ar console.log), lai vizuāli pārliecinātos, ka matemātika strādā.

4. uzdevums: Interaktivitātes pievienošana ar pogu (Padziļināti)

Piesaistīt lietotāja darbību (pogas klikšķi) koda izpildei, izveidojot funkciju un dinamiski mainot spēles stāvokli reālajā laikā.

Izpildes soļi:

  1. Failā index.html (virs skripta taga) pievieno HTML pogu ar notikumu uztvērēju: <button onclick="iegutMonetu()" class="toggleBtn">Savākt monētu</button>.
  2. Failā script.js definē jaunu funkciju, kas izpildīsies, kad poga tiks nospiesta: function iegutMonetu() { ... }
  3. Pārvieto punktu palielināšanas matemātiku un console.log() izvadi uz šīs funkcijas iekšpusi (starp figūriekavām {}).
  4. Saglabā failus, atsvaidzini pārlūku un spied pogu! Pārbaudi, vai konsolē punkti katru reizi palielinās.
  5. Veic Commit un Push uz GitHub ar ziņu "Pievienoti pirmie JS mainīgie, loģika un poga".

Biežākās kļūdas (un kā tās labot):

  • Kļūda konsolē (Uncaught ReferenceError): Tu mēģini izmantot mainīgo pirms tas ir definēts, vai arī failā ir drukas kļūda nosaukumā. JavaScript atšķir lielos un mazos burtus (Punkti nav tas pats, kas punkti)!
  • TypeError: Assignment to constant variable: Tu mēģini mainīt vērtību, kas definēta ar const. Pārveido to uz let, ja vērtībai spēles laikā ir jāmainās.
  • Pārlūkā nekas nenotiek: Pārbaudi savu <script> tagu HTML failā un pārliecinies, ka pogas onclick atribūtā funkcijas nosaukums precīzi sakrīt ar JS failā izveidoto funkciju (arī iekavām () jābūt klāt!).

Koda piemērs

// 1. Mainīgo definēšana
let skolotajs = "Eduards Baumanis";
let palidzejis = 0;
let nokaitinats = 0;
const uzvarasPunkti = 100;

// 2. Sākotnējā izvade konsolē
console.log("Sākam stundu!");
console.log("Stundas sākumā skolotājs palīdzējis: " + palidzejis + " reizes.");

// 3. Vērtību mainīšana (Manuāla simulācija skriptā)
nokaitinats = nokaitinats + 1; // Iegūstam 1 nokāri
console.log("Skolotājs ir nokaitināts: " + nokaitinats + " reizes.");

// 4. Funkcija interaktivitātei (Izsauc ar HTML pogu)
// HTML failā jābūt: 
function prasiPalidzibu() {
    palidzejis = palidzejis + 1; 
    console.log("Skolotājs tikko palīdzēja! Kopā palīdzēts: " + palidzejis + " reizes.");
}
(Šo redzēsi pārlūkprogrammas Console sadaļā, nospiežot F12)
