9. klase 5 tēmas · 30 dubultstundas HTML · CSS · JavaScript Tīmeklis + JS spēles Bezmaksas · MIT licence

Par šo kursu

Šis kurss ir ievads tīmekļa tehnoloģijās un spēļu izstrādē, kas paredzēts 9. klases skolēniem. Visu kursu vieno viens liels mērķis — uzbūvēt un publicēt paša izstrādātu pārlūkspēli.

Skolēni apgūst darbu ar Visual Studio Code un GitHub, veido tīmekļa lapas ar HTML un CSS, pēta lietotāju pieredzes dizainu, tad pakāpeniski iemācās JavaScript programmēšanu, lai pilnveidotu spēles loģiku, animācijas un mijiedarbību.

Kursa noslēgumā skolēns publicē savu spēli internetā ar GitHub Pages un prezentē to saviem klasesbiedriem — tieši tāpat, kā to dara īsti tīmekļa izstrādātāji.

Priekšzināšanas nav nepieciešamas — kurss sākas no nulles.
