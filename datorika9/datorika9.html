Tēmu secība un savienojumi

Katras tēmas prasmes ir pamats nākamajai. Mācīšanās iet no struktūras → stila → loģikas → spēles dzinēja → publicēšanas.

01

Tīmekļa struktūra un dizaina domāšana

Izstrādes vides uzstādīšana (VSCode, GitHub), HTML dokumenta pamatelementi, satura strukturēšana, dizaina pētniecība, UX/UI pamati un spēles koncepcijas izstrāde.

6 dubultstundas HTML · Git · GitHub ↗ Projekts: Spēles projekta pieteikums
↓ HTML pamati → stila apguvi
02

Stils, multimediji un datņu pārvaldība

CSS stila pamati, izkārtojums ar flexbox un grid, krāsu teorija, tipografija, multivides elementi (attēli, video, audio), datņu un mapju organizēšana projektā.

6 dubultstundas CSS · Flexbox · Grid ↗ Projekts: Spēles vitrīnas lapa
↓ vizuāla lapa → interaktīva loģika
03

Algoritmi un JavaScript pamati

Ievads JavaScript programmēšanā: mainīgie, operatori, nosacījumi, cikli, masīvi, funkcijas. DOM manipulācija un notikumu apstrāde (onclick, addEventListener). Spēles loģikas pamati.

6 dubultstundas JavaScript · DOM · Algoritmi ↗ Projekts: Radošā teksta spēle JS
↓ pamata loģika → spēles dzinējs
04

Spēles dzinēja izstrāde un funkcijas

Canvas API un animācija ar requestAnimationFrame, spēlētāja vadīšana, sadursmes noteikšana, spēles stāvokļu pārvaldība (sākums, spēle, beigas), punktu sistēma un ieguvumi.

6 dubultstundas Canvas · Animācija · Sadursmes ↗ Projekts: Spēles dzinēja paplašinājums
↓ spēle gatava → publicēšana internetā
05

Noslēguma projekta izstrāde un publicēšana

Spēles pilnveidošana un atkļūdošana, optimizācija mobilajām ierīcēm, README dokumentācija, publicēšana ar GitHub Pages, prezentācijas gatavošana un vienaudžu vērtēšana.

6 dubultstundas GitHub Pages · README · Prezentācija ↗ Noslēguma projekts: Publicēta spēle internetā
